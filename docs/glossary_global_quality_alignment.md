This pass records those risks through same-topic `confusable_with` metadata rather than changing quiz-generation behavior.
- Mixed Russian/English display conventions: `Case study / анализ случая` remains intentionally bilingual because the repository glossary already used the English term. `Хиндсайт` remains Russian transliteration with aliases; no external-source claim is made.
- Source/provenance gaps: all `question:*` references resolve to approved questions in the same active category. `supplied_snippet:*` entries remain local provenance markers and are not elevated to original-source proof.
- Concepts appearing in more than one topic: basic neural-structure terms intentionally appear in both `fiziologiya_cheloveka` and `psihofiziologiya`; this supports topic-local quizzes and does not create same-topic duplicates.
