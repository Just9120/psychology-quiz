# psychology-quiz

`psychology-quiz` — Telegram-бот викторины по психологии для учебного использования. Текущий продуктовый scope: только **Module 1**. Бот работает в режиме **long polling** (без webhook), **Web UI отсутствует**, внешняя runtime-генерация вопросов (RAG/retrieval) отсутствует.

## Текущий scope (Module 1)

Активные категории:
- Введение в профессию
- Общая психология
- Физиология человека
- Физиология ВНД
- Психофизиология

## Режимы викторины (UX v2)

Поддерживаются три режима запуска:
- `Конкретная тема`
- `Микс из выбранных тем`
- `Все темы`

Краткий flow:
- `Конкретная тема` → выбор одной категории → выбор количества вопросов → выбор сложности.
- `Микс из выбранных тем` → мультивыбор категорий → `Готово` / `Сбросить` → выбор количества вопросов → выбор сложности.
- `Все темы` → выбор количества вопросов → выбор сложности.

Для режима `Микс из выбранных тем` выбранный набор категорий сохраняется на уровне сессии. Сценарий `Пройти еще раз` повторяет тот же selected mix, а не пересобирает темы случайно.

## Вспомогательный UX

- persistent reply keyboard в private chat (`🎯 Начать викторину`, `ℹ️ Помощь`, `👁 Режим чтения`, `🙈 Скрыть меню`)
- ручное скрытие меню через `🙈 Скрыть меню`
- восстановление меню через inline-кнопку `👁 Показать меню`
- fallback-восстановление через `/start`
- режим чтения: `Обычный` / `Бионическое чтение`

## Data flow: JSON → seed → SQLite

- Source of truth для question bank: JSON в `content/questions/module1/`.
- SQLite **не** является источником истины: это runtime/storage layer.
- Заполнение и обновление БД выполняется сидером `scripts/seed_questions.py`.
- Операционная цепочка обновления контента:
  1. изменить JSON;
  2. сделать PR;
  3. после merge запустить seed;
  4. runtime SQLite получает обновленные данные.
